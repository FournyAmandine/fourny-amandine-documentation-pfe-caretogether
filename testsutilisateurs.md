# Tests utilisateurs 


Voici les différentes vidéos faites pour les test utilisateurs. J'ai fait tester à 3 potentiels utilisateurs 5 tâches : 
- Se créer un compte et ajouter une annonce
- Marquer cette annonce comme vendue
- Filtrer les annonces disponibles
- Choisir une annonce et contacter le vendeur
- Enregistrer une annonce et la retrouver sur son profil


## Utilisateur 1 - Anne-Catherine - Bandagiste  
Pour le premier test utilisateur, j'ai demandé à une bandagiste. CareTogether vise ce public professionnel, c'est donc une personne suceptible d'utiliser ce site car ça rentre dans ses fonctions.
